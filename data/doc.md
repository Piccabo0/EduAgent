# 摩擦力的基本定义
两个相互接触并挤压的物体，当它们发生相对运动或具有相对运动趋势时，就会在接触面上产生阻碍相对运动或相对运动趋势的力，这种力叫做摩擦力。摩擦力通常用“f”来表示。

# 摩擦力产生的物理本质
摩擦力的产生本质上是因为物体的接触面并非绝对光滑（即使肉眼看起来平整，在显微镜下观察也会发现存在凹凸不平的结构）。当两个物体相互接触时，当物体发生相对运动时，这些凹凸不平的表面相互挤压在一起时会彼此阻碍，从而形成摩擦力。

# 摩擦力的分类
摩擦力可以分为静摩擦力，滑动摩擦力和滚动摩擦力。

## 静摩擦力
两个相互接触的物体，当其接触表面之间有相对运动的趋势，但仍然保持相对静止时，就会在接触面上产生阻碍这个相对运动趋势的力，这个力就是静摩擦力。

## 滑动摩擦力
两个相互接触的物体，当其接触面之间产生相对运动时，接触面间会产生阻碍它们相对运动的力，这个力就是滑动摩擦力。滑动摩擦力的方向与相对运动的方向相反。

## 滚动摩擦力：
两个相互接触的物体，当其接触面之间产生相对运动且相对运动属于滚动时，产生的摩擦力叫滚动摩擦力。

# 摩擦力的方向
静摩擦力的方向与相对运动趋势的方向相反。滑动摩擦力的方向与相对运动的方向相反。

# 摩擦力的典型案例
在生活中，草地上滚动的足球逐渐停下，滑冰运动员停止蹬冰后慢慢减速，以及汽车刹车后很快停稳，都是摩擦力在起作用的典型例子。

# 滑动摩擦力的测量方法
测量滑动摩擦力通常需要使用弹簧测力计。测量时，必须用弹簧测力计沿水平方向拉动木块，并确保木块在水平长木板上做匀速直线运动。在这种匀速直线运动的状态下，弹簧测力计显示的拉力数值就等于木块受到的滑动摩擦力的大小。这一测量过程实际上是利用了二力平衡的原理，将难以直接测量的摩擦力转化为了可读的拉力数值。

# 影响滑动摩擦力大小的因素
滑动摩擦力的大小与接触面的粗糙程度与接触面上的压力有关。当接触面的粗糙程度一定时，接触面受到的压力越大，产生的滑动摩擦力就越大。当接触面受到的压力一定时，接触面越粗糙，滑动摩擦力就越大。

## 探究不同因素对滑动摩擦力的影响的方法
通常采用控制变量法。通过保持其他变量不变，每次只改变压力或接触面粗糙程度中的一个变量，便可以找到各因素与摩擦力之间的定量关系。

# 增大摩擦的主要方法
在需要增大摩擦力的场合，人们通常采取增大接触面受到的压力、增大接触面粗糙程度或变滚动摩擦力为滑动摩擦力等方法。

## 增大摩擦力的例子
自行车刹车时用力捏闸是通过增大压力来增大摩擦。体操运动员在手上涂防滑粉则是通过增大接触面的粗糙程度来增大摩擦。

# 减小摩擦力的主要方法
减小摩擦的方法包括减小接触面受到的压力、减小接触面的粗糙程度、用滚动摩擦力代替滑动摩擦力或者使接触面彼此分离。

## 减小摩擦力的例子
给门轴合页加润滑剂可以形成油膜使表面不直接接触从而减小摩擦。气垫船通过压缩空气将船体与水面隔开，以达到大幅减小摩擦的目的。中国古代劳动人民通过滚木来移动巨石或巨木是通过变滑动摩擦力为滚动摩擦力的方式减小摩擦。现代机器中广泛应用的滚动轴承也是基于变滑动摩擦力为滚动摩擦力的方式减小摩擦，其内圈与外圈之间装有光滑的钢球或钢柱。当轴承转动时，钢球或钢柱在内外圈之间滚动，这种滚动摩擦远小于滑动摩擦。
